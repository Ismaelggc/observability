Sintaxis básica

Curso de sintáxis básica de Markdown.

Encabezado

Podemos tener encabezados h1 o h2 dependiendo si usamos # o ##. Cómo podemos ver al principio tenemos un encabezado h1.
Mientras que el segundo es un encabezado h2.

Negritas y cursiva

Para usar negritas tenemos que escribir dentro de ** nuestro texto ** y para cursivas usamos * nuestro texto *. También podemos escribir
en negritas y cursivas usando *** nuestro texto ***. Prueba de negrita y cursiva.

Código

Si queremos escribir código usaremos las comillas invertidas. Vamos a poner un ejemplo de como nos moveríamos entre directorios en Linux usando cd ...

Si quisieramos usar código que incluya comillas invertidas tendríamos que usar comillas invertidas dobles MYSQL_ROOT_PASSWORD=`DevPass123!`

También podremos crear bloques de código usando comillas invertidas triples.

cd Documentos;  
ls -a;  
mkdir Nueva Carpeta

En este ejemplo, podemos ver como crea una visualización del bloque de código.

Enlaces

Para crear enlaces usamos los corchetes [] y los paréntesis(). En los corchetes escribimos el texto que aparecerá y en los paréntesis el hiperenlace. Este sería un ejemplo para Google.com.

También puede usarse para dirigirnos a secciones del propio texto. Sintaxis básica.

Esto puede usarse para correos electrónicos usando < > ismaelgutierrez1990@gmail.com. Ademáse se pueden crear enlaces a una url más rápidamente https://www.google.com.

Títulos a los enlaces

Puedes agregar un título al enlace agregando un comentario al final del mismo entre comillas dobles. Este aparecerá si pasamos el ratón encima del enlace Google.

Formato enlace

Al igual que con el texto normal, podemos agregar formato a los enlaces usando las mismas marcas * ** ***. Podemos ver que el enlace a Google ahora aparece en negrita y cursiva.

Enlaces a referencias

Esto es un tipo especial de enlace consta de dos elementos que van entre corchetes. El primero muestra el texto que se va a mostrar y el segundo es el identificador de referencia a enlazar. Aquí hemos creado una Referencia usando Google.com.

Imágenes

Para usar imágenes tenemos que usar el signo de ! seguido de corchetes para el texto de la imagen y comillas para su url (ya sea online o una ruta de nuestro equipo).

Podemos agregar un título a la imagen incluyéndolo entre comillas después de escribir la ruta de la imagen.
Por lo que nuestro código quedaría así:

![ nombre de la imagen ] ( ruta "título")

Imagen con enlace

Para agregar un enlace a la imágen tenemos que rodear el código de Markdown con corchetes y agregar un enlace entre paréntesis

Quedando nuestro código de la siguiente forma:

[ ! [ nombre de la imagen ] ( ruta "título") ] (enlace)

Listas

Para agregar una lista tenemos que agregar un 1 seguido de un punto y un espacio. Aunque el orden de los indices no es diferencial, siempre tiene que empezar por el número 1.

  1. Primero
  2. Segundo
  3. Tercero

Esta es la lista numerada, Podemos crear una lista usando astericos o guiones, pero se deben mantener a lo largo de la lista y no ir cambiado en cada elemento.

  • Primero
  • Segundo
  • Tercero

También podemos anidar listas usando cuatro espacio o le tabulador.

  1. Primero
  2. Segundo
    • Primero
    • Segundo

También se pueden añadir elementos e imágenes en las listas, para ello tendremos que dejar un espacio entre líneas en blanco e incluir lo que queramos.

  1. Primero

  2. Segundo

Código HTML

Podemos agregar diréctamente código HTML ya qué la mayoría de editores y procesadores de Markdown soportan HTML. Sin embargo siempre es interesante consultar el manual para cada caso específico.
Podemos resaltar un texto en negritas usando ** palabra ** o usando <strong>palabra</strong>. Hemos conseguido visualizar el código gracias a que hemos usado \ delante del mismo.

Tablas

Aquí vamos a crear tablas usando el símbolo | y los guiones para definir la cabecera de las columnas.
Si queremos crear una rabla quedaría algo así: 

| Cabecera1| |Cabecera2|
| -------- | | ------- |
| Elem 1 , 1 | | Elem 1, 2 |
Nombre Apellido
Ismael Gutierrez

Usaremos el doble punto en el separador de la cabecera para indicar dónde queremos alinear la tabla. Si queremos alinear a la izquierda usaremos :---- , para centrado :------: y para la derecha -----:.

Una vez señalada dónde quieres hacer la nota a pie de página tendremos que escribir el código seguido de dos puntos y esta referencia aparecerá en el pié de página.

Resaltado de sintaxis

La mayoria de editores Markdown soportan el resaltado de código. Para ello tenemo que indicar el lenguaje después de la triple comilla inversa de apertura de un bloque de código.

const value = 3
let result = value * 4;

Ids de cabeceras

Podemos identificar los encabezados en Markdown usando el atributo id. Para añadir este atributo tenemos que usar un identificador personalizado entre llaves. El identificador tendrá que empezar por el carácter #. Un ejemplo para nuestro código quedarí así:

## Cabecera {#id-encabezado}

Este encabezado puede ser enlazada tanto en el documento Markdown como en otras páginas creando un enlace normal y usando el id del encabezado quedando así

[Encabezado](#id-encabezado)

Esto depende de cada editor, en mi caso estoy usando Marker por lo que al no ser compatible con esta instrucción he tenido que hacerlo mediante código html <h3 id="encabezado">Encabezado</h3>.
